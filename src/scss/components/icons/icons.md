# Icons

## Generic syntax

```html
<i data-cssinfo class="Icon Icon--wisembly"></i>
```

## Sizes

```html
<i data-cssinfo class="Icon Icon--like Icon--s"></i>
<i data-cssinfo class="Icon Icon--like Icon--l"></i>
<i data-cssinfo class="Icon Icon--like Icon--xl"></i>
```
